## ‼️ 규칙
1. 개인 repo를 생성한다. ```Algorithm-Name```
2. 본인의 repo 로 fork하여 사용한다.
3. fork한 repo를 백준허브에 연동한다.
4. commit을 한 후에 PR을 날린다. 이때, ```이름/월.일 제출``` 형식으로 통일한다.
5. 개인 repo에서 merge한다.
